# Adding categories and requirements

---

To start adding categories and requirements to an entity, first select the entity you wish to work on and then click the “Add Catgeory” button on the Client page.

![](/assets/wp-content/uploads/2025/06/Screenshot-2025-09-21-at-5.47.20-pm.png)

This opens the Add Category window.

![](/assets/wp-content/uploads/2025/06/add-category-modal-1024x450.png)

Select the categories you wish to add from the dropdown. Then click “Save”.

![Screenshot of selected categories](/assets/wp-content/uploads/2025/06/add-category-selections-1024x454.png)

These categories will now be added to the Client’s entity and displayed on the page.

![Screenshot of client page](/assets/wp-content/uploads/2025/06/client-page-with-entities-1024x294.png)

Note that you can also add a custom category using the Add Category form. When added to the entity it will be empty – you will then have to add your own requirements to the Category.
