# Client page

When a client logs in, they are taken to their Client page. The page is divided into three columns.

The left column shows which period the Client is reviewing, and which period is the Active Period. Below that section are the taxable entities (Individual, Company or Trust) for which a Client needs to submit a tax return in that Period.

The progress bar below the Period selection dropdown shows how much the Client has completed for that Period. The progress indicator reflects the number of requirements the Client has sent back for review from the combined entities in the selected (not the Active) period.

Each Entity has a list of Categories, shown in the middle column. Each category has a set of [requirements](/knowledge-base/requirements-for-clients/) (right column) that the Client needs to respond to. You can see how many requirements in each category have been completed in some text just below each Category title e.g. “1/6 requirements completed”.(“completed” means the Client has added a response and/or some attachments to a requirement and sent it back to the Accountant to review).
