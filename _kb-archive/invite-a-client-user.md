# Invite a Client user

---

A Client can have one or more users. For example the Client has two individual entities, and so each individual needs their own access.

To add a client user, first open the [Manage Client](/knowledge-base/manage-client-page/) page for that Client.

![Screenshot of the Manage Client button](/assets/wp-content/uploads/2025/06/manage-client-button.png)

On the Manage Client page, click the “ADD USER” button.

![Screenshot of Add User button](/assets/wp-content/uploads/2025/06/add-user-button-1024x84.png)

Complete the fields on the “Add or invite user” popup. By default the invite is not sent immediately unless you check the “Send invite email” checkbox. This option is there in case you still need to do some more setup for the client, or you want to [send multiple invites at once](/knowledge-base/invite-multiple-client-users-at-once/).

![](/assets/wp-content/uploads/2025/06/add-or-invite-user-modal.png)

Clicking “Save” will close the popup and display the new user on the Manage Client page.

If the invite was sent the date the invite will expire is displayed, as well as the status of the invite.

![](/assets/wp-content/uploads/2025/06/manage-client-table-1024x245.png)

The invitation will be sent to the new user, explaining how to confirm their user account and login to **taxy**. A copy of the email will be cc’d to the Accountant.
