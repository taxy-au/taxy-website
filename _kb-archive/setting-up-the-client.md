# Setting up the client

---

Follow these steps to set up a client to use **taxy**:

[Add the new client](/knowledge-base/adding-a-client/) – you will only need the client name to do this.

[Add one or more entities](/assets/wp-content/uploads/2025/06/add-entity.png) – a client may have one more more taxable entities, for example an individual return, and a trust.

[Add categories and requirements](/assets/wp-content/uploads/2025/06/add-category.png) to each entity – each catalogue type has a set of categories and requirements appropriate for that entity type.

[Add and invite the client user](/knowledge-base/invite-a-client-user/) – once you have created the entities and added the appropriate categories for each one you can go ahead and invite the client user.
