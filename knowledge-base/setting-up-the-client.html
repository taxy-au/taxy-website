Setting up the client

Follow these steps to set up a client to use taxy:

Add the new client – you will only need the client name to do this.

Add one or more entities – a client may have one more more taxable entities, for example an individual return, and a trust.

Add categories and requirements to each entity – each catalogue type has a set of categories and requirements appropriate for that entity type.

Add and invite the client user – once you have created the entities and added the appropriate categories for each one you can go ahead and invite the client user.
