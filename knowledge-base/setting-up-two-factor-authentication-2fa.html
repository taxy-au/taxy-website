Setting up two-factor authentication (2FA)

taxy offers two-factor authentication (2FA) for all users. It is not mandatory, but we recommend all users take the option.

New users are asked to set up 2FA when they first login.

You can choose not to set up 2FA and if you are absolutely certain you don’t want it, you can click the “Don’t ask me again” checkbox. If you clicked “Yes” you will be taken to a simple setup wizard.

Enter your mobile phone number (Australian only) and follow the verification steps.

2FA can be turned off or on at any time by going to the user page (click the user icon in the top navigation bar).

On the user page click the enable/disable button and follow the instructions.
