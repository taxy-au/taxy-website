Working Papers

Once a client has submitted all their responses and documents, and the accounting team have checked them all for completeness and marked them “Ready for Return”, you can generate a set of Working Papers as a PDF or spreadsheet for download. You can work from the Working Papers to finalise the client’s tax return.

The PDF Working Papers will include every category and requirement, along with every attachment uploaded by the client.

The Excel Working Papers file has a sheet for each category, as well as a summary sheet.

You can also download the PDF and Excel versions, along with all the attachments, in a single zip file.

Note that spreadsheets are not included in the PDF Working Papers (due to formatting issues) and need to be downloaded separately.

To generate Working Papers for an entity, simply select one of the options from the Working Papers button just below the Client name.
